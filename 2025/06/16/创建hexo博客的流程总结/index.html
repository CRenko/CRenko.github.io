创建hexo博客的流程总结

使用hexo在github上创建博客的流程

第一步 本地安装hexo

首先hexo是一个基于node.js的框架，所以安装hexo之前需要先安装node.js
，关于node.js的安装就不过多说明了，有用的人肯定都会装。所以跳过安装node.js
直奔hexo的安装。

首先 在git或者cmd上安装hexo

npm install -g hexo

等待安装完毕hexo就算安装上了，然后我们需要找一个位置新建空文件夹，这个空文件夹就是本地博客的存放位置，然后在空文件夹里打开git

输入项目初始化创建指令

hexo init

这时候等待初始化完成，原本的空文件夹就会初始的hexo，然后使用网页创建指令

hexo g

这样就会根据source文件夹里的md文件生成对应的网页文件了，再使用项目预览指令

hexo s

这时候完成的话会出现以默认4000的端口的预览地址，直接访问这个网址就能看到hexo生成的网页了。

第二步 github创建仓库

这一步很简单，在自己的github上创建一个名为用户名.github.io的仓库以创建自己的网站。

在通过ssh和本地电脑连接之后可以把项目上传

ssh-keygen -t rsa -C "github的邮件地址"

这个命令生成ssh文件，然后在~/.ssh里找到 rsa_id.pub复制里面的内容到github上绑定一下就可以了。

第三步 配置hexo的上传github功能

在hexo的项目根目录下有一个叫_config.yml的文件，这个文件里的最下端有一项deploy，默认是空的，把它的内容改为

deploy:
  type: git
  repository: git@github.com:github用户名/github用户名.github.io.git
  branch: main

然后再在git上使用安装命令安装hexo-deployer-git

npm install hexo-deployer-git --save

只有安装了这个才能用命令一键上传hexo项目到github上

然后使用上传命令

hexo d

等待上传完毕就可以打开https://你的用户名.github.io来查看你的博客了
